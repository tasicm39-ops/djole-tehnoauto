# djole-tehnoauto

Sajt za Đole Tehnoauto (statički HTML/CSS).

## Javni pregled (GitHub Pages)

Posle uključivanja Pages-a u repozitorijumu, sajt je na:

**https://tasicm39-ops.github.io/djole-tehnoauto/**

### Kako uključiti GitHub Pages

1. Otvori repozitorijum na GitHubu → **Settings** → **Pages** (levi meni).
2. Kod **Build and deployment** → **Source**: **Deploy from a branch**.
3. **Branch**: `main`, folder **`/ (root)`** → **Save**.
4. Sačekaj 1–2 minuta; stranica će biti na adresi iznad.

Kada kasnije povežeš sopstveni domen, u istom odeljku možeš dodati **Custom domain** i onda u kodu zameniti bazni URL u `canonical`, `sitemap.xml` i `og:image` na novi domen.
